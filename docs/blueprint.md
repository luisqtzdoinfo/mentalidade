## Core Features:

- Quiz Engine: Dynamically render quiz questions and answers.
- Progressive UI: Display confetti animations on answer and progress bar on quiz page

## Style Guidelines:

- Primary color: Dark gray (#333333) for a modern and serious look.
- Background color: Light gray (#F0F0F0) to provide contrast and clean feel.
- Accent color: Electric Blue (#7DF9FF) for highlights and call-to-action buttons to create visual impact and a sense of energy.
- Body and headline font: 'Inter' for a modern, neutral feel.
- Minimalist icons for quiz sections and results display.
- Clean and modern layout with a focus on readability and easy navigation.
- Subtle confetti animation on correct answers and loading animations for smooth transitions.
